##1. Project Overview

    This project analyzes ecommerce transaction data to identify
    customer purchasing behavior, revenue trends, product performance,
    and marketing effectiveness.

##2. Dataset Overview
- 1200 rows
- 14 columns
- Ecommerce transaction dataset

##3. Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

##4. Data Cleaning Summary

- Converted Date to datetime format
- Handled missing CouponCode values
- Verified absence of duplicate transactions
- Performed data validation checks

##6. Key Findings

- TotalPrice distribution is strongly right-skewed.
- High-value orders were identified as legitimate bulk purchases.
- Chairs and Printers generated the highest sales.
- Instagram was the leading referral source.
- June generated the highest monthly revenue.
- Customer C38840 was the highest-spending customer.

##7. Recommendations

- Increase stock levels before June.
- Expand inventory for high-performing products.
- Reward high-value customers through loyalty programs.
- Increase focus on successful Instagram campaigns.
